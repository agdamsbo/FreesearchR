# Add sparkline to datagrid

Add sparkline to datagrid

## Usage

``` r
add_sparkline(
  grid,
  column = "vals",
  color.main = "#2a8484",
  color.sec = "#84EF84"
)
```

## Arguments

- grid:

  grid

- column:

  clumn to transform

## Value

datagrid

## Examples

grid <- mtcars |>
  default_parsing() |>
  overview_vars() |>
  toastui::datagrid() |>
  add_sparkline()
grid
